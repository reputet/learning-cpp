# C++ experiments

The diagrams are drown by one of the following commands depending on the tool used:

```bash
diagrams flowchart diagram.flowchart flowchart.svg
mmdc -s 2 -i diagram.mmd -o flowchart.png
```

Thanks to the
1. [flowchart.js.org](https://flowchart.js.org/) project and the corresponding [diagrams](https://github.com/francoislaberge/diagrams/) CLI tool;
2. [mermaid](https://mermaid-js.github.io/mermaid/) project and the appropriate [mmdc](https://github.com/mermaid-js/mermaid-cli) CLI tool.
